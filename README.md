# Slitherlink Solver

This repository contains the implementation for my bachelor [thesis](https://github.com/Omarr-Alkhatib/Slitherlink-Solver/releases/download/v1.0/Abschlussarbeit_464143.pdf):

**Comparative Evaluation of Constraint Programming and Deductive Search for Solving Slitherlink Puzzles with Difficulty Assessment**

## Overview

This project implements different approaches for solving [Slitherlink](https://en.wikipedia.org/wiki/Slitherlink) puzzles. The main focus is on CP-SAT based solvers and a Deductive Search solver that emulates human-style reasoning and can be used to assess puzzle difficulty.

Implemented solvers:

- `SolverCPVerifier.java`: Generates solutions satisfying the local constraints and verifies whether the result forms a valid single loop.
- `SolverCPLazy.java`: Detects invalid loop structures and excludes them iteratively by adding cuts.
- `SolverCPSingleLoop.java`: Encodes the single-loop constraint directly in the CP-SAT model.
- `SolverDS.java`: Solves puzzles using Deductive Search with logical rules and bounded hypothetical reasoning.

## Requirements

- Java 17
